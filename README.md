# 学习 Pull request

### Pull request

- fork仓库

- 修改代码并提交和推送

- github 上点击 Pull request

- 创建`pull request`

![new pull request](picture/new%20pr.png)

```
1.New pull request
2.Create pull request (进入填写信息页面)
3.填写信息
4.Create pull request (创建)
```
